# Telco-Churn-Analysis---Amdox-Internship

# 📊 Customer Churn Prediction and Sales Dashboard

## 🚀 Project Overview
This project uses **Data Analytics and Machine Learning** to analyze customer behavior, predict churn probability, and visualize sales trends using Power BI.

The goal is to help businesses **reduce customer churn**, improve **retention strategies**, and **maximize revenue** through data-driven insights.

---

## 🎯 Objectives
- Predict customers likely to churn
- Analyze sales and revenue trends
- Segment customers based on behavior
- Provide actionable business insights through dashboards

---

## 📁 Dataset
We used the **Telco Customer Churn Dataset** from Kaggle:  
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

---

## 🧠 Machine Learning Models Used
- Logistic Regression  
- Random Forest  
- K-Nearest Neighbors (KNN)

These models are trained on customer data to predict **churn probability**.

---

## ⚙️ Technologies Used
- **Python** (Pandas, NumPy, Scikit-learn)  
- **Power BI** (Dashboard & Visualization)  
- **Jupyter Notebook / Google Colab**

---

## 🔄 Workflow
1. Data Collection  
2. Data Preprocessing (cleaning, encoding, scaling)  
3. Model Training & Evaluation  
4. Churn Probability Prediction  
5. Export results to CSV  
6. Visualization using Power BI  

---

## 📊 Dashboard Features
- Customer churn analysis  
- High-risk customer identification  
- Revenue trends & forecasting  
- Customer Lifetime Value (LTV)  
- Retention strategy simulation  

---

## 📈 Key Insights
- Customers with higher monthly charges and shorter tenure are more likely to churn  
- Contract type and payment method significantly affect churn  
- Identifying high-risk customers helps improve retention strategies
